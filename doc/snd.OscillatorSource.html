Class: OscillatorSource

snd. OscillatorSource

任意の波形を再生するオシレータ音源を扱うクラスです。
snd.OscillatorSource.SINEなどの定数値でサイン波・矩形波・のこぎり波・三角波を設定できる他、波形はPeriodicWaveクラスでも定義が可能です。

new OscillatorSource(id)

新しくオシレータ音源を生成します。
Parameters:
Name Type Description
id String この音源をあらわすID
Properties:
Name Type Description
frequencyParam AudioParam このOscillatorの周波数を変更するためのAudioParamです。他のAudioUnitの出力をこのOscillatorの周波数の値に渡す場合などに使用するためのもので、このプロパティに直接値を代入することはできません。
Oscillatorに具体的な周波数を設定したい場合は、frequencyプロパティを使用してください。
oscillatorが初期化されていない場合、undefinedが戻ります。
frequency Number このOscillatorの周波数です。
frequencyParamとは異なり、数値を直接渡すことが可能です。
単位は[hz]です。
oscillatorが初期化されていない場合、undefinedが戻ります。
detumeParam AudioParam このOscillatorの周波数を変更するためのAudioParamです。他のAudioUnitの出力をこのOscillatorの周波数の値に渡す場合などに使用するためのもので、このプロパティに直接値を代入することはできません。
Oscillatorに具体的な値を設定したい場合は、detuneプロパティを使用してください。
oscillatorが初期化されていない場合、undefinedが戻ります。
detune Number このOscillatorのdetuneです。
frequencyに対して、音程の微妙な調整を行いたい場合に使用する値です。
detuneParamとは異なり、数値を直接渡すことが可能です。
単位は[cent]です。（cent … 1/1200オクターブ。半音の1/100）
oscillatorが初期化されていない場合、undefinedが戻ります。
