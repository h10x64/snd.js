Class Index

_global_

snd

snd.jsの基幹ネームスペースです。

snd.AudioDataManager

AudioBufferの管理を行うクラスです。
データの読み込みだけでなく、複数のAudioBufferのローディングを待機するときなどに使用してください。

snd.AudioUnit

1つのオーディオユニットを定義するクラスです。 インタフェースクラスなので、継承されることを前提としています。 引数にAudioUnitを要求するメソッドに渡すオブジェクトは、ここで定義されている各メソッドを実装している必要があります。

snd.BufferSource

バイナリデータを再生する音源です。
詳細はWebAudioAPIの仕様を参照してください。
どのブラウザも、基本的にwav形式のファイルには対応していますが、mp3については対応状況がまばらです。

snd.GainOnlyUnit

主ボリュームのみの単純なユニットです。

snd.Listener

リスナを表すクラスです。 AudioContext#Listenerをラップしています。 setListenerメソッドを呼び出すまではthis.listenerはnullで、実際の出力へは反映されません。 this.listenerがnullである間も位置は記録されるので、setListenerメソッドが呼び出された時点で WebAudioAPIのListenerに、そのオブジェクトで設定された姿勢が反映されます。

snd.MediaElementAudioSource

HTMLのメディアタグを音源として使用する音源クラスです。
詳細はWebAudioAPI仕様を参照してください。

snd.MediaStreamAudioSource

ストリームを音源として使用する音源クラスです。
詳細はWebAudioAPI仕様を参照してください。

snd.OscillatorSource

任意の波形を再生するオシレータ音源を生成します。
詳細は、WebAudioAPIの仕様を参照してください。

snd.PosDir

位置と向きをあらわすクラスです。 位置を表すposベクトル、正面向きを表すdirベクトル、上方向を表すupベクトルの3つのベクトルで位置と向きを管理します。

snd.SoundEnvironment

シミュレーションする音環境を表すクラスです。
リスナーやユニットなどを管理します。
さまざまなユニットは最終的にSoundEnvironment#connectAudioUnitメソッドを使って、実際の出力へ反映されるようになります。

snd.SoundNode

PannerNodeを使用するパンニングに対応したユニットです。

snd.srctype

音源の種類をあらわす値を入れるネームスペースです。

snd.status

音源のステータスを表す値を入れるネームスペースです。

snd.vec3

3次元ベクトルクラスです。 球座標としても使われ、その場合、x, y, zの値はそれぞれ x: 方位角 y: 仰角 z: 距離 として扱われます。
