Class: MediaElementAudioSource

snd. MediaElementAudioSource

HTMLのメディア要素（Audioタグなど）を音源として使用する音源クラスです。
使用するタグに id が設定されている場合は、JSON.stringify メソッドを使用した際にその id が出力されるようになります。

new MediaElementAudioSource(id, htmlMediaElement)

新しくメディアタグを使用する音源を生成します。
Parameters:
Name Type Description
id String この音源のID
htmlMediaElement HTMLMediaElement HTMLのメディアタグ要素

setLoop(doesLoop)

この音源をループ再生するかどうかを設定します。
Audioタグを使用するため、ループ終点からループ始点に戻る速度がBufferSoundSourceと比較して遅く、間にブレイクが入る可能性があります。
なめらかにループしたい場合はBufferSoundSourceの使用を検討してください。
Parameters:
Name Type Description
doesLoop type ループ再生するか否か
