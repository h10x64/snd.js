Class: BufferSource

snd. BufferSource

AudioBufferを使用してバイナリデータを再生する音源です。
詳細はWebAudioAPIの仕様を参照してください。
wav, mp3などが再生可能ですが、ブラウザにより対応状況が異なります。

          start(when, offset, duration)

          この音源の再生を開始します。
          一時停止はできません。
          start()とすると、すぐにデータの頭から終わりまでの再生が開始されます。
          Parameters:
          Name Type Description
          when Number 何秒後に再生を開始するか
          offset Number 音源の再生開始位置（単位:秒）
          duration Number 音源の再生終了位置（単位:秒）
