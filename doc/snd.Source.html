Class: Source

snd. Source

各種音源クラスの親クラスとなる抽象クラスです。
start, stopなどの抽象メソッドは継承する子クラスで実装してください。

new Source(id)

音源を生成します。
typeプロパティはsnd.srctype.NONEに
statusプロパティはsnd.status.NONEに
それぞれ設定されます。
Parameters:
Name Type Description
id String この音源のID
Properties:
Name Type Description
isSource Boolean このオブジェクトがsnd.Sourceであることを表すプロパティです。
volumeParam AudioParam この音源の音量を設定するためのAudioParamです。他のAudioUnitの出力をこのOscillatorの周波数の値に渡す場合などに使用するためのもので、このプロパティに直接値を代入することはできません。
この音源の音量に具体的な数値を設定したい場合は、volumeプロパティを使用してください。
volume Number この音源の音量を取得・設定するプロパティです。
単位は[倍]です。（デシベルではありません）
type String この音源のクラス名です。
値を設定することはできません。
status snd.status この音源の状態を表すプロパティです。
値を設定することはできません。
