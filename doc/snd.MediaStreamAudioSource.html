Class: MediaStreamAudioSource

snd. MediaStreamAudioSource

音声ストリームを音源として使用する音源クラスです。
WebRTCのGetUserMediaで取得したストリームを使用することができます。

new MediaStreamAudioSource(id, mediaStream, callback, オブジェクトの生成に失敗した時に呼び出されるエラーコールバックです。(未設定可)
)

新しくストリーム音源を作ります。
Parameters:
Name Type Description
id String この音源のID
mediaStream MediaStream 再生するデータストリーム
nullの場合、自動で音声ストリームを取得します。
callback function オブジェクトの生成に成功した時に呼び出されるコールバックメソッドです。(未設定可)
呼び出される際は、引数として生成されたオブジェクトが渡されます。
mediaStreamがnullの場合のみ使用される引数で、mediaStreamに値が設定されている場合は使用されません。
オブジェクトの生成に失敗した時に呼び出されるエラーコールバックです。(未設定可)
 function 呼び出される際は、getUserMediaメソッドから返されるエラーオブジェクトが渡されます。
mediaStreamがnullの場合のみ使用される引数で、mediaStreamに値が設定されている場合は使用されません。
未設定の場合、コンソールへのログ出力のみ行います。
