Class snd.Listener

リスナを表すクラスです。 AudioContext#Listenerをラップしています。 setListenerメソッドを呼び出すまではthis.listenerはnullで、実際の出力へは反映されません。 this.listenerがnullである間も位置は記録されるので、setListenerメソッドが呼び出された時点で WebAudioAPIのListenerに、そのオブジェクトで設定された姿勢が反映されます。
Defined in: snd.js.
